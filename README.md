# 🔒 Password Shield

A sleek, modern, and cyber-themed **Password Strength Analyst** application built using Python and Tkinter. This tool evaluates the security level of your passwords in real-time, checking for length, uppercase letters, numbers, and special symbols with an interactive UI.

---

## ✨ Features

- 🕵️ **Real-time Analysis:** Validates your password instantly as you type.
- 🎨 **Modern Cyber-Theme UI:** Designed with a beautiful deep midnight-blue aesthetic mimicking a glassmorphism card overlay.
- 📈 **Dynamic Strength Bar:** An animated progress bar that changes color based on score entropy (Red for Weak, Yellow for Medium, Cyan for Strong).
- 👁️ **Toggle Visibility:** Easily hide or show your password with a smooth custom button.
- 📋 **Live Checklist:** Tracks which security rules have been met dynamically.

---

## 🛠️ Requirements

- **Python 3.x**
- **Tkinter** (Pre-installed with standard Python distributions)
